Intruksi Fungsi
ls untuk mengetahui semua isi folder
pwd untuk mengetahui posisi folder/git yang aktif
cd change directory / pindah directory ( ganti folder )
git init Buat inisiasi folder menjadi repo
git config --global user.name "username github" inisiasi nama untuk commit ( disarankan sama dengan github
git config --global user.email "email@.com" inisiasi email untuk commit ( disarankan sama dengan github
git status mengetahui status file yang masih dalam tahap change ( belum up ke staged )
git add . menambahkan semua file yang sudah di edit ke staged ( agar siap commit )
git add nama.file menambahkan file tertentu yang sudah di edit ke staged ( agar siap commit )
git commit untuk commit
git commit -m "pesan commit" -m adalah message ( commit dengan menambahkan pesan )
git log melihat history commit kita
git log -3 melihat history 3 commit terakhir kita (angka dapat disesuaikan)
git log -- nama.file melihat history commit kita pada file tertentu (sesuaikan nama file)
git checkout xxxxx mengembalikan koondisi ke waktu commit terakhir (xxxxx = 5 digit pertama id commit )
git checkout xxxxx -- nama.file mengembalikan koondisi ke waktu commit terakhir (hanya pada file tertentu saja, nama.file disesuaikan)
